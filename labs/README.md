# ECPEL Labs

Labs are evidence-scoped exercises used to validate repository concepts without overstating implementation maturity.

> **Principle:** No evidence, no implementation.

## Lab Index

| Lab | Purpose | Status |
| --- | --- | --- |
| [Terraform Local Foundation](terraform-local-foundation/README.md) | Local-only Terraform workflow for variables, validation, locals, modules, outputs, tests, and evidence handling | Simulated |

## Status Notes

- **Simulated** means the lab is a local-only, non-production demonstration and must not be described as deployed infrastructure.
- This lab must not be marked **Implemented** or **Validated** without successful command evidence.
- Labs must not require cloud credentials unless explicitly documented and approved in a future PR.
